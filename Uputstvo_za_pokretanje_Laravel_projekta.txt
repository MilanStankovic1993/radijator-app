
Uputstvo za pokretanje Laravel projekta sa Git-a

1. Kloniraj projekat sa Git repozitorijuma
   git clone <URL_REPOZITORIJUMA>
   cd <IME_PROJEKTA>

2. Instaliraj zavisnosti sa Composer-om
   (Uveri se da imaš instaliran PHP i Composer)
   composer install

3. Kopiraj .env fajl i podesi osnovne parametre
   cp .env.example .env
   Otvori .env fajl i podesi:
   - DB_CONNECTION=mysql
   - DB_HOST=127.0.0.1
   - DB_PORT=3306
   - DB_DATABASE=ime_baze (kreiraj praznu bazu u MySQL-u sa ovim imenom)
   - DB_USERNAME=korisnik
   - DB_PASSWORD=lozinka

4. Generiši aplikacioni ključ
   php artisan key:generate

5. Pokreni migracije i seedere (ako ih ima)
   php artisan migrate --seed

6. Pokreni lokalni razvojni server (opciono)
   php artisan serve

7. Pristupi aplikaciji u browseru
   Idi na adresu: http://localhost:8000

Napomena:
- Baza mora biti kreirana pre pokretanja migracija.
- Ako projekat koristi Filament, može se napraviti korisnik komandom ako nije seed-ovan:
  php artisan make:filament-user
